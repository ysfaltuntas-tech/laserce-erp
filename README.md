# Laserce ERP

Laserce Metal için modüler iş takip, stok, müşteri ve teklif uygulaması.

## Kurulum

1. Supabase projesinde `supabase/schema.sql` dosyasını çalıştır.
2. `.env.example` dosyasını `.env` olarak kopyala.
3. Supabase URL ve anon key değerlerini gir.
4. `npm install`
5. `npm run dev`

## Vercel değişkenleri

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
